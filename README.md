# University Management System (ERP) 🎓

A complete full-stack ERP system developed to manage university operations. It features a secure backend API and dedicated portals for Administrators, Faculty, and Students.

## 🚀 Key Features
* **Admin Dashboard:** A desktop GUI built with CustomTkinter for admissions, fee management, and database control.
* **Faculty Portal:** A secure web interface for professors to manage course grading and view student rosters.
* **Student Portal:** A web interface for students to check timetables, view transcripts, and generate fee challans.
* **Admissions Portal:** An online application form with live merit score calculation.

## 🛠️ Tech Stack
* **Web Frontend:** HTML, CSS, JavaScript
* **Desktop Frontend:** Python (CustomTkinter)
* **Backend:** Python (Flask API)
* **Database:** MySQL

## 📁 Main Files Overview
* `admin_dashboard.py` - The Desktop Admin Panel.
* `api_server.py` - The backend API connecting the database to the portals.
* `university database.sql` - The complete SQL script to instantly set up the database and tables.
* `login.html`, `faculty.html`, `student.html`, `apply.html` - The web interfaces.
